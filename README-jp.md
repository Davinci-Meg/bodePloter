# ボードプロッター

ボードプロッターは、信号の周波数応答を視覚化するためのツールです。

## インストール

以下の手順に従って、ボードプロッターをインストールしてください。

1. 依存関係をインストールします。

    ```bash
    npm install
    ```

2. プロジェクトをビルドします。

    ```bash
    npm run build
    ```

3. ボードプロッターを起動します。

    ```bash
    npm start
    ```
